Privacy Policy

Last updated: February 2026

What data we collect

When CodeSentry analyzes a pull request, it accesses the following data from your GitHub repository:

  • Repository metadata (name, owner)
  • Pull request diffs (added and modified code)
  • Commit messages associated with the pull request
  • GitHub installation and webhook event metadata

What data is sent externally

Pull request diffs are sent to Anthropic's API (Claude) for behavioral analysis. This allows CodeSentry to generate summaries and identify potential risks in the code changes. Anthropic's privacy policy is available at anthropic.com/privacy.

No other third-party services receive your code.

Data retention

Analysis results (findings, risk scores, and LLM summaries) are stored in our database. No raw source code is stored after analysis completes. Only structured analysis results and metadata are retained.

How to uninstall and delete your data

You can uninstall CodeSentry at any time from your GitHub organization or account settings. To request deletion of all stored analysis data, uninstall the GitHub App and email privacy@codesentry.dev.

Changes to this policy

We may update this privacy policy from time to time. Changes will be reflected on this page with an updated date.
