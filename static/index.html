CodeSentry

AI code review for AI-generated code

CodeSentry is a GitHub App that automatically reviews pull requests for issues in AI-generated code. It combines Semgrep static analysis with Claude-powered behavioral review to catch bugs, security issues, and anti-patterns that slip through human review.

Install on GitHub
